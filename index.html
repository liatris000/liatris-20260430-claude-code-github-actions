GitHub IssueにClaude Codeを連携。タスクを書くだけでブランチ作成・実装・PR作成まで自動化します。

issue: opened / labeled

Issue → PR 自動実装

Issueに claude ラベルを付けるか、本文に /claude-implement を書くと自動実装が始まります。

🔍
PR comment: @claude review

PR 自動レビュー

PRのコメントに @claude review と書くと、バグ・セキュリティ・パフォーマンスの観点でレビューします。

🔧
PR comment: @claude fix

レビュー指摘を自動修正

レビュー後に @claude fix と書くと、指摘事項をすべて修正してコミットします。

🚀 セットアップ手順

  1. このリポジトリをフォーク or テンプレートとして使用
  2. リポジトリの Settings → Secrets and variables → Actions を開く
  3. 以下のシークレットを登録:
ANTHROPIC_API_KEY Anthropic コンソールで取得
GITHUB_TOKEN Actions が自動生成（追加不要）

  1. Issueを作成して claude ラベルを付けて試してみましょう！
